Equipo 1

  HMW: ¿Cómo podríamos alcanzar un nivel alto de comodidad y fidelidad de los usuarios para poder destacar de las
                                            otras casas de cambio?

       IDEA FINAL: Implementar beneficios a nuestros usuarios en base a un sistema de puntos, con un
 requerimiento estándar de digitalización, fácil de entender, que luego de acumular cierta cantidad de puntos se
                     pueden canjear a cambio de un beneficio respecto al tipo de cambio.
¿Qué queremos probar con el prototipo a la hora hacer el test con sus usuarios?
- Queremos probar que el cliente a través de un sistema de canje de puntos se volverá fiel a la plataforma, ya que recibirá beneficios
constantes en el tipo de cambio. Asimismo, se creará una sensación de comodidad, puesto que este nuevo sistema tiene un requerimiento
estándar de digitalización ameno e intuitivo. Por lo que, a través de los beneficios y la comodidad el usuario se volverá un cliente constante y
fiel de la plataforma.
¿Cuál es el objetivo de hacer el prototipado?
- Realizar una simulación del sistema de canje de puntos, con la finalidad de ver cual es la respuesta de los usuarios y recibir feedback. Es
decir, se creará una especie de maqueta en la que se probará el flujo de interacción para después evaluar e identificar qué funciona y que no
para poder arreglarlo, y así poder lanzarlo a la plataforma y que tenga éxito.
¿Qué deseamos conocer o aprender de ese prototipo y mostrárselo a sus clientes?
1. Si el sistema de canje de puntos produce un sentimiento o sensación en el usuario que lo motive a usarlo otra vez.
2. Si el requerimiento estándar de digitalización es lo suficientemente simple e interactivo para que el usuario lo pueda usar sin problemas y no
se frustre.
3. Si el usuario entiende los beneficios de este nuevo sistema en el tipo de cambio y si confía en este para usarlo a la hora de hacer una
transacción.
¿Con quiénes vamos a probar nuestro prototipo?
- Con los usuarios de Midpoint, ya que ellos ya conocen la plataforma y saben cómo usarla. Por lo que, con ellos podremos verificar si es que
este nuevo sistema es fácil de usar y si produce los sentimientos o sensaciones esperados.
- Con los nuevos usuarios que se vean interesados en hacer su primer cambio con dólares, ya que con las “coins” podrá tener descuentos si es
activo con la plataforma.

Equipo 3

  HMW: ¿Cómo podríamos facilitar el uso de Midpoint para que los clientes puedan transferir con diferentes servicios
                                    bancarios para tener mayor accesibilidad?

 IDEA FINAL: Compatibilidad de cuentas bancarias con los bancos más conocidos donde sea posible realizar
 las transacciones, y conexión directa vía WhatsApp con la persona encargada de realizar la transacción para
                                           solucionar cualquier duda.
¿Qué queremos probar con el prototipo a la hora hacer el test con sus usuarios?
Primeramente, podríamos ver la satisfacción de los usuarios al mostrarles nuestro prototipo. Además de la eficiencia ante cualquier problema
que pueda suscitarse. Por último, la cantidad de nuevos clientes que atraeremos.
¿Cuál es el objetivo de hacer el prototipado?
El objetivo del prototipado es verificar el diseño realizado y confirmar que cuenta con las características específicas planteadas para resolver el
problema del usuario.
¿Qué deseamos conocer o aprender de ese prototipo y mostrárselo a sus clientes?
Principalmente, queremos saber si nuestra idea inicial les parece funcional y util. Si harian algun cambio, ya sea porque no funcione
adecuadamente u otro motivo.
¿Con quiénes vamos a probar nuestro prototipo?
Las pruebas se realizaron tanto con los usuarios como con posibles usuarios tambien se probaria con los mienbros de Midpoint FX para saber
si a ellos les es útil.
