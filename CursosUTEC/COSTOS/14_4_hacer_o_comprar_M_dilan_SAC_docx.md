Hacer o comprar

02/09/2021

MCG

**CASO: M’DILAN SAC**

M’dilan SAC fabrica papel kraft para usar en uno de sus productos principales (bolsas ecológicas). La producción normal anual son 100,000 planchas de este tipo de papel. El costo por 100 planchas es como sigue:

| Material directo                       |          | \$260.00     |
|:---------------------------------------|:---------|:-------------|
| Mano de obra directa                   |          | \$100.00     |
| Costos indirectos de fabricación       |          |              |
| Variables                              | \$120.00 |              |
| Fijos                                  | \$160.00 |              |
| Total Costos indirectos de fabricación |          | \$280.00     |
| **Costo total para 100 planchas**      | ** **    | **\$640.00** |

Industrias Paper ofreció vender a M’dilan 100,000 planchas que necesitará durante el próximo año, a US\$600 por 100 planchas. Si M’dilan acepta la oferta de Industrias Paper, las instalaciones usadas para fabricar el papel kraft podrían utilizarse en la producción de papel tissue. Este cambio ahorraría a M’dilan US\$90,000 en costos relevantes. Además, se eliminaría un elemento de costo de US\$100,000 incluido en los costos indirectos de fabricación fijos específicamente relacionado con la fabricación del papel kraft.

Pregunta:

¿Debería M’dilan aceptar la oferta de Industrias Paper?
