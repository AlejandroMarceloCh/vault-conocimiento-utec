        Ejercicio 4:
        Implementar página web
-   Cree un nuevo bucket “gcolchado-miweb”
-   Suba o cargue la web simple o la web
    plantilla con index.html en la raíz
-   Luego marque todos los directorios y
    archivos y “Hacer público”
-   Configure “Alojamiento de sitios web
    estáticos” (Pestaña “Propiedades”) con
    index.html
-   Visualice la página web por el navegador.
    Ejemplo:
                                                http://gcolchado-miweb.s3-website-us-east-1.amazonaws.com
