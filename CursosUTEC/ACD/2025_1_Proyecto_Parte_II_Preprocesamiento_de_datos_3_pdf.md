 Semestre 2025-1

                               Proyecto Final
                   DS3021 Análisis Computacional de Datos

 Parte II: Preprocesamiento de Datos

I.    Objetivo:

      ● Aplicar técnicas computacionales para el preprocesamiento de datos con la
        finalidad de asegurar la calidad de los datos.

II.   Instrucciones:

      1. Para este proyecto usaremos como input los datos recolectados en la primera
         parte del proyecto.
      2. Aplique las técnicas de preprocesamiento trabajadas durante el curso. No es
         necesario aplicar todas.
      3. Redacte un reporte técnico de las técnicas de preprocesamiento utilizadas para
         preprocesar el dataset.

 Un ejemplo de una posible estructura del reporte técnico podría ser:
    1. Data Integration
    2. Data Cleaning
          a. Nivel I
          b. Nivel II
          c. Nivel III: Análisis de Valores Faltantes
          d. Nivel III: Análisis de Outliers
    3. Data Transformation

 III. Entregables y Deadlines

      ● Deadline: 27 de noviembre de 2025
      ● Entregables:
            ○ Dataset .csv
            ○ Script .ipynb (documentado)
            ○ Reporte técnico .pdf
      ● El peso de esta segunda parte de su proyecto es de 30% de la nota final de
        Proyecto P.
