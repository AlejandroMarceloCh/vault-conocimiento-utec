### Nanoeconomía

Mientras que la macroeconomía se enfoca en el comportamiento
y el desempeño de economías enteras, y la microeconomía
examina a los agentes o entidades individuales,

existe la necesidad de introducir un nivel más detallado de teoría
económica para identiﬁcar y codiﬁcar las tendencias de
comportamiento y desempeño predecibles de las entidades
individuales, ya sean humanas o dispositivos.

Este concepto innovador se conoce como nanoeconomía

**Figura:** Texto centrado en tipografía sans-serif gris con términos clave resaltados en cian/azul claro: «macroeconomía», «microeconomía», «de las entidades individuales, ya sean humanas o dispositivos» y «nanoeconomía». Tres párrafos: el primero contrasta macroeconomía y microeconomía; el segundo introduce la necesidad de un nivel más detallado de teoría económica; el tercero presenta el concepto de nanoeconomía.

NANOECONOMIA
La nanoeconomía es la teoría económica de las propensiones de
comportamiento y desempeño predichas de entidades individuales
(humanas o dispositivos).

La nanoeconomía es un término para describir la teoría económica
de aprovechar la IA para descubrir las propensiones de
comportamiento y desempeño predichas de humanos y
dispositivos (o conocimientos analíticos) que están enterrados en
los datos de compromiso del cliente y gestión operativa de la
organización.

Improving

## Data monetization práctico
### Grupos proyecto

Cada grupo debe desarrolle una estrategia de monetización de datos para su proyecto simulada utilizando los tres enfoques:

- **Improving :** Identificar cómo los datos pueden ser utilizados para mejorar las operaciones internas, reducir costos y aumentar la eficiencia.
- **Wrapping :** Proponer características o experiencias adicionales basadas en datos que aumenten la propuesta de valor para el cliente.
- **Selling :** Desarrollar ideas sobre cómo los datos recopilados pueden ser comercializados como productos o servicios de información.

## Glosario

**Value Engineering Competency** — Competencia que significa comprender cómo las organizaciones pueden aprovechar los datos (big data) y la analítica avanzada (IA/ML) para crear valor.

**Economía** — La rama del conocimiento que se ocupa de la producción, el consumo y la transferencia de riqueza o valor.

**Nanoeconomía** — La teoría económica de las propensiones de comportamiento y desempeño predichas de entidades individuales (humanas o dispositivos). Término para describir la teoría económica de aprovechar la IA para descubrir las propensiones de comportamiento y desempeño predichas de humanos y dispositivos (o conocimientos analíticos) que están enterrados en los datos de compromiso del cliente y gestión operativa de la organización.

**Analytic Profiles / Analytic Scores** — Codifican, comparten, reutilizan y refinan continuamente las propensiones de comportamiento y desempeño predichas de los activos humanos y de dispositivos clave de la organización.

**Monetización de datos** — La generación de retornos financieros a partir de activos de datos; convertir los datos en dinero.

**Proceso datos-insight-acción** — El proceso de creación de valor a partir de datos; ocurre cuando personas o sistemas usan datos para desarrollar insights que informan la acción, lo que genera valor.

**Improving** — Enfoque de monetización de datos que genera dinero cuando las organizaciones utilizan datos para cambiar la economía del trabajo para mejor y luego eliminan o redirigen el margen resultante.

**Wrapping** — Enfoque de monetización de datos que genera dinero cuando las organizaciones utilizan características o experiencias impulsadas por datos para mejorar la propuesta de valor de un producto y luego aumentan los precios o venden más productos.

**Selling** — Enfoque de monetización de datos que genera nuevos ingresos cuando las organizaciones comercializan datos en forma de una solución de información.
