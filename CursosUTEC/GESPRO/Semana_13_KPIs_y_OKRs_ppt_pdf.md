   Key Performance Indicators

              ● El indicador clave de rendimiento o desempeño, como su propio
                nombre indica, es una medida que sirve como herramienta para
                valorar el nivel de rendimiento de un proceso.
              ● Al establecer KPIs, podemos reflejar el estado actual de un
                proceso con respecto a un área concreto, y a partir de ahí, actuar
                para optimizar las estrategias.
              ● Es por este motivo, que un indicador siempre debe tener un valor
                meta (principal combustible para mejorar).

 Características de KPIs
    Modelo S.M.A.R.T. este hace referencia a que los indicadores deben
    ser “inteligentes, posibles y eficaces”.
Gestión de Proyectos                                     Gabriela Pella




                       Specific (específico):
                       Debe definirse de forma clara y
                       concisa.
Gestión de Proyectos                                      Gabriela Pella




                       Measurable (medible):
                       Debe ser factible establecer una
                       métrica para cuantificar los
                       resultados.
Gestión de Proyectos                                      Gabriela Pella




                       Attainable (alcanzable):
                       Debe estar acorde (realista) a
                       los recursos y tiempo disponible
                       del negocio.
Gestión de Proyectos                                       Gabriela Pella




                       Relevant (relevante):
                       Debe aportar valor al negocio, su
                       cálculo debe significar una
                       iniciativa de mejora importante.
Gestión de Proyectos                                    Gabriela Pella




                       Timely (acotado en el tiempo):
                       Definido dentro del tiempo en
                       el que se va a ejecutar.
