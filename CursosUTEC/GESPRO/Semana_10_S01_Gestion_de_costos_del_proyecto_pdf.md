   Ejemplo ilustrativo

 Se planificó un proyecto con un nivel de calidad bajo y se ahorró 20,000
 dólares. Después de que el proyecto se completó, los costos de
 mantenimiento ascendieron a 125,000 dólares, en lugar de 15,000 que
 pudieron haber costado si se consideraba una calidad de más alto nivel.

 ¿Se tuvo un ahorro real? ¿Por qué?
 Rpt: Con esto, entendemos que los 20,000 dólares de “ahorro”, se terminaron convirtiendo en un costo de 110,000 dólares.

    Conceptos clave
             ● Análisis de valor: buscar una forma menos costosa de
               hacer el mismo trabajo.
             ● Costos hundidos: dinero ya gastado, sobre el cual no se
               puede recuperar.
             ● Capital de trabajo: cantidad de dinero que la empresa
               posee para invertir.
             ● Depreciación: registrar como gasto una parte del costo
               del activo fijo, a lo largo de su vida útil.

  Estimación análoga
                         Técnica basada en la utilización de datos históricos de una actividad o
                                                     proyecto similar.
                            Conocida también como estimación top-down, se utiliza cuando:
                                     (1) Se dispone de poca información del proyecto.
                               (2) El proyecto presenta similitudes con un proyecto previo.
                       (3) Las personas encargadas de realizar la estimación conocen el trabajo a
                                                          estimar.




                                                                   Recuperado de http://blog.quizpm.com/como-estimar-el-coste-de-un-proyecto
Gestión de Proyectos                                                                                                                      Gabriela Pella




  Estimación paramétrica
                          Esta técnica utiliza una relación estadística entre datos históricos
                        relevantes y otras variables para calcular una estimación del costo del
                                                 trabajo del proyecto.
                       Con esta técnica se pueden conseguir niveles superiores de exactitud, en
                                      función de los datos que utilice el modelo.




                                                              Recuperado de http://blog.quizpm.com/como-estimar-el-coste-de-un-proyecto

 Estimación ascendente
                       Conocida como estimación bottom-up, se utiliza cuando se dispone de
                                         información detallada del proyecto.
                       El costo de cada paquete de trabajo o actividad se calcula con el mayor
                                               nivel de detalle posible.
